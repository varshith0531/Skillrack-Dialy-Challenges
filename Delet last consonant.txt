Delete Last Consonant The program forms a singly linked list with the given N alphabets. Then the program removes the last occurring consonant from the linked list and prints the remaining N-1 alphabets as the input. Your task is to implement the function deleteLastConsonant so that the program runs successfully. Note: There will be at least one consonant in the singly linked list. Example Input/Output 1: Input: 6 p e o p l e Output: p e o p e Explanation: The last occurring consonant is 'l' and hence it is deleted from the singly linked list.
